CD de distribution de JSesh (version @JSESHVERSION@)

Bonjour, ce CD contient une version de JSesh, éditeur de hiéroglyphes en logiciel libre.

Pour installer JSesh, votre machine doit disposer d'un interpréteur java. C'est le cas de tous les Macintosh sous MacOSX. Par contre, si vous avez un PC sous windows, vous trouverez plusieurs versions de java dans le répertoire JRE. Je vous suggère de lancer j2re-1_4_2_06-windows-i586-p.exe.

Il vous suffira ensuite de double-cliquer sur JSesh-install.jar

Le CD contient un certain nombre de textes lisibles par JSesh dans le répertoire Textes.

Si le logiciel d'installation ne fonctionne pas, vous pouvez toujours copier le contenu du répertoire sauvegarde sur votre disque dur. Il contient une version de JSesh que vous pouvez lancer en cliquant sur sauvegarde/JSesh/libs/jsesh.jar

JSesh est un logiciel libre distribué sous la licence LGPL. Vous pouvez le copier, et le distribuer librement. S'il vous est utile, vous êtes cordialement invité à choisir un (ou plusieurs !) textes, à les saisir dans JSesh, et à me les expédier à l'adresse rosmord@iut.univ-paris8.fr. Je les intégrerai dans la base de textes. Quand JSesh sera doté d'un éditeur de signes, vous pourrez aussi proposer de nouveaux hiéroglyphes.

Vous pouvez aussi, et c'est même fortement conseillé, me faire part de vos observations, des fonctionnalités que vous jugeriez intéressantes, des bugs que vous observez, etc.
