
# JSesh Fonts License

Most of the glyphs included in jseshGlyphs-....jar have been created by Serge Jean Paul THOMAS.

According to his instructions:

- it is allowed to distribute the fonts with JSesh
- it is allowed to use the fonts when JSesh was used as a library
- it is allowed to use JSesh fonts for publications (paper, databases, web sites).
- it is allowed to modify the signs for the same reasons and use them under the same terms.

M. Thomas  doesn't want the font to be used for other hieroglyphic word processors.

Please note that this can't be changed. I am aware that this might be a bit inconvenient for some projects.


The few signs drawn by Serge ROSMORDUC are distributed according to the SIL FONT LICENCE. In some cases, the information is available in the SVG file itself.
