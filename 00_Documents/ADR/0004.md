# ADR 3 : Saving User Hieroglyphic folder along with other font information

Status
: accepted

## Context

The current code had a specific processing of the Hieroglyph folder preferences, which was different from the one use for standard fonts.
It's difficult to follow.

## Decision

Load and save hieroglyph folder preferences along with other informations.

## Consequences

We might have to change the initialisation code. Or to be happy with having a redundant loading of this information.
