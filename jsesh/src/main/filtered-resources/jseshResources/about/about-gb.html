About JSesh ${project.version}

JSesh is a free hieroglyph editor, written by S. Rosmorduc (serge.rosmorduc@qenherkhopeshef.org). A huge thank to M. Serge J.P. Thomas, sš jqr ḏbꜣ.w, who has drawn most of the signs of the JSesh fonts.

JSesh home : http://jsesh.qenherkhopeshef.org

Library used in JSesh

  • JSesh uses the application framework of JHotDraw 7, by Werner Randelshofer (http://www.randelshofer.ch/oop/jhotdraw/index.html), which provides a nice, Mac-Friendly, document-oriented system, and is usable for lots of projects, even those which don't deal with vector graphics.
  • IText, a very complete pdf library for java (http://www.lowagie.com/iText/) ; distributed under either the LGPL or the Mozilla public license.
  • IzPack for its installer : http://www.izforge.com/izpack/, available under the GPL, but the distributed software itself can have any license.
  • JLex for the lexical analyser : http://www.cs.princeton.edu/~appel/modern/java/JLex/ . License : see directory licenses
  • JCup for the parser. http://www.cs.princeton.edu/~appel/modern/java/CUP/. License : see directory licenses
  • The quaqua look and feel library for the mac provides a better mac user experience. It's made by Werner Randelshofer (http://www.randelshofer.ch/quaqua/) and distributed under the LGPL license.
  • The great form layout library, available from http://www.jgoodies.com/downloads/libraries.html, under the BSD open source license.
